# 🏏 Cricket Match Analyzer Pro

## 🌟 About the Project
This is a smart Python application designed to analyze a 6-ball cricket over. It takes ball-by-ball inputs from the user and generates a complete match summary. 

I built this project to master **Loops, Conditionals, Counters, Accumulators, and Exception Handling** in Python.

## 🚀 Key Features
- **Ball-by-Ball Tracking:** Records runs scored on every ball.
- **Smart Calculations:** Automatically calculates Total Runs, Total Balls, Fours, Sixes, Boundaries, and Dot Balls.
- **Strike Rate Engine:** Accurately calculates the batter's strike rate.
- **🛡️ 100% Crash-Proof (Error Handling):** I manually found and fixed **17 bugs**! The system uses `try...except` blocks to handle invalid string inputs (like typing "abcd" instead of numbers), ensuring the program never crashes.

## 🧠 Logic & Concepts Used
- `for` loops and `Tuples/Lists` for iteration.
- `if-elif` blocks for checking boundaries and dot balls.
- Accumulators (`total_runs += runs`) and Counters (`balls += 1`).
- Advanced `try...except` for robust input validation.
